T1-git-blog

This week I learned all about git and github at DBC. Both are useful version control tools for web developers. I’ll do my best to clearly share what I’ve learned with you here. Bear with me now! I’ll start with version control. Version control is a system for storing and sorting different iterations of a project. This is helpful because it allows multiple people to be working on their own copies of the same files. It also allows users to save older versions of code which they can revert back to, if needed. Basically, if a mistake is made that breaks the project there are earlier, working versions of the program.
